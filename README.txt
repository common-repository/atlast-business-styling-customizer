== Description ==

This is a styling companion plugin of the free [Atlast Business WordPress Theme](https://wordpress.org/themes/atlast-business/). It adds eight new sections in the customizer with over 72 color settings. With this plugin you can definately style the Atlast Business theme very easily and with no coding skills. You can style the
1. Homepage section (and all the sub-sections)
2. Top Bar section
3. Header / Navigation section
4. Header Image section
5. Page/Posts/Archives elements
6. Sidebar Widgets
8. Footer area and footer widgets 
9. Copyright area
10. Default button styles

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'asbc.zip' to the `/wp-content/plugins/` directory
2. Unzip the folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Open the WordPress customizer by going to Admin->Appearance->Customize
5. A new "Atlast Business Styling" panel is shown on top.
6. Enjoy!

== Frequently Asked Questions ==

= Should I have installed the Atlast Business theme in order to use this plugin? =

Definately yes. However you may still use it if you use another theme and you can hire a developer to made some code adjustments on your theme.
